# AI 英語練習句產生器 - 提示詞與格式指南

本指南提供了一個現成的 AI 提示詞模板（System Prompt）。您可以直接將其複製並發送給 ChatGPT, Claude 或 Gemini，讓 AI 為您產生完全符合本特訓器匯入格式的英文句子與中文翻譯。

---

## 📋 建議的導入數據格式

本特訓器支援兩種匯入格式，建議使用**「格式 A（單斜線分隔）」**，因為它最容易閱讀且最容易在 AI 聊天介面中直接複製。

### 格式 A：單斜線分隔（推薦，極簡流暢）
每行一句，英文與中文之間用空一格的斜線 ` / ` 分隔：
```text
I need a prescription. / 我需要一張處方。
Does this medicine have side effects? / 這種藥物有副作用嗎？
Take this medicine after meals. / 飯後服用這種藥。
```

### 格式 B：標準 JSON 格式（適合開發者或批次處理）
```json
[
  {
    "english": "I need a prescription.",
    "chinese": "我需要一張處方。"
  },
  {
    "english": "Does this medicine have side effects?",
    "chinese": "這種藥物有副作用嗎？"
  }
]
```

## 🤖 AI 提示詞模板 (Prompt Template)

請複製下方虛線內的提示詞，將其發送給 AI，並根據您的需求替換 `[括號]` 中的內容：

---

**請為我生成英語學習練習題。**

**主題要求：**
- 主題：`[例如：Unit 12 飯店入住與退房 / Unit 13 機場英語]`
- 難度：`[例如：初級 / 中等 / 高級]`
- 句子數量：`[例如：10 句]`

**輸出格式規範：**
1. 每一行只包含一句英文句子與中文翻譯。
2. 英文與中文之間必須用 ` / `（前後有空格的斜線）隔開。
3. 英文句子中的標點符號請使用標準半形符號（如 `.`、`?`、`!`、`,`）。
4. 中文翻譯請使用繁體中文。
5. 不要輸出任何其他的解釋文字、引導語或序號，直接輸出句子列表即可。

**輸出範例：**
We would like to check in, please. / 我們想要辦理入住。
Is breakfast included in the room rate? / 房價有包含早餐嗎？
What time is check-out tomorrow? / 明天幾點退房？

---

## 💡 AI 使用小技巧
* **控制句子長度**：如果您的練習偏向初學者，可以在提示詞中加入：「請限制句子長度在 5 到 8 個單字之間。」
* **特定文法練習**：您可以指定文法，例如：「請多使用『現在完成式』或『被動態』來造句。」
* **專有名詞定義**：若有特定職場情境（例如：半導體設備、電子商務），請直接在主題欄寫明，AI 會產出非常道地的專業英文。
